Introduction

BB4 Theme Builder is the easy way to build themes for Button Builder 4. BB4 Theme Builder will take your button images (.psd .tiff .png) and generate the preview, it will also package it up and name the files correctly.

Using Help

Click an item on the left to see help about the topic. To find answers to your questions, type a word or phrase in the box above, then press Return.

Once the theme is finished, you can post it over at the Realmac Software forum to share with other users.

Happy Theme Building!
