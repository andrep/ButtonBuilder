Feature Outline

This chapter give you a brief outline of the features available.

Themes
The Themes Drawer will allow you to browse and choose a suitable theme for your button. It is located on the left hand side of the main window by default.

Click the preview of each button in the theme drawer to update the button in the main window. If the themes drawer is not open, press the "Themes" toolbar item.

Title
The Title sheet allows you to set the title, color, position and effects on the type that appears on top of the button.

Click the Title toolbar icon to change the text on the button in the main view.

Font
Button Builder uses the standard Mac OS X Font panel.

Click the Font toolbar icon to change the font size and style of the text that appears on the button.

Options
The Options sheet allows you to change the color and opacity of the button and also the background color.

Click the options sheet to change:

Button Opacity: Move the slider to alter the transparency of the button.

Color Overlay: Click the color swatch to choose a new color for your button.

Background Color: Click the color swatch to choose a new color for the background.

Export
Button Builder supports two kinds of image, PNG and JPEG.

Click Export and choose an export option. If you require the button to be on a transparent background use PNG. If you require your button to have a background color use JPEG.

If you are unsure what to use then JPG is a safe choice, as it has better support for older browsers.
