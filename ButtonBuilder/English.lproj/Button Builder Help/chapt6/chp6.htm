Version History

A complete list of the changes and updates made to Button Builder since the first public release of version 3 in 2002.

Version 4.1 February 1st 2007

Released as open source software!

Version 4.0.2 May 26th 2004

Fixed image display bug, whereby the image would appear blurred.
Export Options moved to save panel to fit in with Apple HIG.

Version 4.0.1 April 30th 2004

Export Button to JPG.
Background color supported.
Documentation improved.
Apple Help added.
Theme drawer resize bug fixed.

Version 4.0 April 20th 2004

Completely Re-written in Cocoa
New Improved Interface
Over 50 new buttons
Smooth Text Shadows
Color Overlay Feature
Smooth Scaling
Export Button to PNG (including transparency)

Version 3.3.2 April 23rd 2003

New registration system.
Various minor bug fixes.
Fixed external java script error.
Fixed Crash on quit bug under Jaguar. (again)

Version 3.3.1 December 1st 2002

Fixed external java script error (added missing </script> tag)
New javascript rollover code. (Thanks to Dave Taylor of Intuitive Systems)
You can now specify the name of the external .js file.
Update file menu layout, so it's easier for new users.

Version 3.3.0 November 16th 2002

Theme mixer added, you can now mix and match any themes.
Select All command added, for text use with text fields.
Improved and updated fonts window.
Font size can now be updated live via a slider.
JavaScript is now saved in an external file (java.js)
Theme window now has live resize installed.
Text effects code improved.
Export Images button added to website dialog.
About box now displays the registered users name.
Crash on Quit bug under Jaguar fixed.

Version 3.2.0 September 1st 2002

File menu updated.
Help tags added to all interface elements.
Build website dialog updated.
More Export options added in "Build Website" dialog
JPEG quality settings now saved.
Remove double preferences menu bug.
Text is now correct size to fit in with Apples aqua GUI guidelines
Save All Images option removed in the Web Export dialog due to bug.
OS X 10.2 Save sheet bug fixed.
Updated manual.
Over 10 other minor improvements.

Version 3.1.0 August 1st 2002

Preferences window layout greatly improved.
Save All Images option added in the Web Export dialog.
Export Alpha Channel option added (great for Adobe Photoshop™ users)
Software update now remembers when you last checked for updates.
Button Manager window is now fully re-sizable.
Button Manager list can now be viewed at two different sizes.
Option to turn off/on CSS in Web Export dialog.
CSS export bug fixed.
Mouse cursor now updates inside the canvas area.
Greatly updated manual.
Manual now includes a tutorial on using Button Builder extreme with Adobe GoLive™

Version 3.0.0 July 1st 2002

First public release.
