FAQ & Tips

Themes
The Themes Drawer will allow you to browse and choose a sutible theme for your button. It is located on the left hand side of the main window by defualt.

Tip: Color Overlay

To get the best result from the color overlay tool we have found it best to set the opacity to around 40% and to make the color as pure and bright as possible.

Tip: Transparency

When Button Builder exports buttons it uses the .PNG format, so it keeps all the transparency attributes. This is a great for web designers and photoshop users alike.

Q. Can I export my buttons to a gif file?

A. No, Unisys has a patent on the LZW compression method used in creating a GIF file. Without paying Unisys a large fee, Button Builder extreme cannot use that method of compression.

Q. The button preview sometimes appears blurry, why is this?

A. This is due to the way Mac OS X handles drawing images, we are currently trying to find a fix for this problem. This side effect will not effect the final output.
