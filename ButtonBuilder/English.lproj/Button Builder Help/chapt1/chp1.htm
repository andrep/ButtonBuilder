QuickStart Guide

This chapter shows you how to create and export a button.

STEP ONE: Choose your Style

Launch Button Builder and open the themes drawer (it should already be open). You can browse the themes by clicking on the preview until you find one that you like. You should notice the main button in Button Builder updates to reflect your choice in realtime.

STEP TWO: Change the Title

Click on the "Title" toolbar button, and a sheet will pop down letting you change the title of your button. This sheet also contains options for text color, shadow and vertical font positioning. As you can see below we have chosen to set the title to "E-mail Me".


STEP THREE: Set the Size

Use the width slider at the bottom of the main window, so that it fits the title you entered.

STEP FOUR: Modify the Color

If your not quite happy with the color of the button you have chosen you can modify it by using the Color Overlay feature. This can be found in the "Option" sheet. To get the best result from the color overlay tool we have found it best to set the opacity to around 30% or 40% and to make the color as pure and bright as possible.

STEP FIVE: Export

If you are happy with the way the button looks you are now ready to export it. Just click the "Export" toolbar button and choose a save destination. Button Builder will then save it in PNG format (including transparency). It's now ready to be used in Photoshop, on the web or in a multimedia project.
